windows快捷键

发表于2023-09-12|更新于2023-09-12|windows
|字数总计:173|阅读时长:1分钟|阅读量:|评论数:

Ctrl+Shift+Esc打开任务管理器

Ctrl+F4 Win资源管理器中切至地址栏；媒体播放中停止。

Ctrl+a 全选All

按Home（定位到行首）然后按Shift+Dnd（行尾）
或者
按End(定位到行尾)然后按Shift+Home
或者
按Home（定位到行首）然后按Shift+↓

  • 按下 “Ctrl + L” 或 “F6” 可以选中地址栏，然后输入搜索关键字。

  • 按下 “Ctrl + K” 或 “Ctrl + E” 可以选中搜索框。

浏览器

Ctrl+Shift+N 进入无痕浏览模式

Ctrl+T 打开新标签

Ctrl+Tab焦点向下一项移动

Ctrl+Shift+Tab=Shift+Tab焦点向上一项移动

Ctrl+F4 删除标签页

win+E 打开文件管理器
